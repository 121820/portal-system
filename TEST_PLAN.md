## 测试环境
- **前端**: Vue 3 + Element Plus + WangEditor
- **后端**: Spring Boot API (假设)
- **存储**: NAS `/vol1/1000/Photos/portal-system`
- **浏览器**: Chrome, Firefox, Edge 最新版本

## 测试执行

执行 `npm run test:unit` 进行单元测试
